# Brand image drop zone

Drop AI-generated images here with these exact names — the app picks them up
on the next reload, no code changes needed:

- `logo.png` — square, transparent background, ~512×512. Replaces the built-in
  SVG mark in the nav and on the login page.
- `dashboard-hero.png` — wide (~1200×400), transparent or white background.
  Shown at the top of the dashboard; hidden while the file doesn't exist.
